# Mehdi Rebahi - Évaluation Finale

## Description

Application web de gestion de bibliothèque développée avec React, Express et MySQL.

## Technologies utilisées

* React
* Vite
* Express.js
* Node.js
* MySQL
* Axios

## Installation Backend

```bash
cd server
npm install
node server.js
```

## Installation Frontend

```bash
cd client
npm install
npm run dev
```
